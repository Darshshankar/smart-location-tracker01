# 📍 Smart Location Tracker

A mobile application built using React Native for real-time location tracking.

## 🚀 Features
- Real-time location tracking
- Google Maps integration
- Firebase database support
- User-friendly interface

## 🛠️ Tech Stack
- React Native
- Firebase
- Google Maps API
